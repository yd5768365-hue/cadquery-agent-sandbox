# CAE智能体使用指南

## 快速启动

在命令行中输入 `cae` 即可激活CAE数字孪生项目智能体。

```bash
cae
```

## 智能体简介

这是一个专门为CAE数字孪生项目提供支持的智能助手，旨在帮助您：

1. **完善项目功能** - 分析代码架构，提供改进建议
2. **规划学习路径** - 制定个性化学习计划
3. **解答专业问题** - 提供CAE理论知识支持
4. **指导项目开发** - 确保代码质量和理论支撑

## 智能体能力

### 理论知识
- 材料力学：应力应变分析、强度理论
- 弹性力学：广义胡克定律、本构方程
- 有限元方法：离散化原理、形函数、单元刚度矩阵
- 计算力学：数值方法、收敛性分析

### 技术技能
- CAD软件：SolidWorks建模与装配
- 网格生成：Gmsh参数化网格控制
- 仿真求解：CalculiX输入文件编写
- 可视化：PyVista/Plotly结果展示

### 编程能力
- Python：FastAPI/Celery/Streamlit
- 容器化：Docker/Kubernetes
- 数据处理：NumPy/SciPy/Pandas
- 机器学习：参数优化与预测
